# Drug Safety repository revision note

This repository snapshot was text-reviewed for consistency with the Drug Safety manuscript package. The README and selected public-facing script comments were revised to remove prior journal-specific wording and to align the repository description with the manuscript title, pharmacovigilance-informed cardiac safety prioritization, strict external comparative evaluation, and Online Resource 1.

No analytical data files, trained weights, model code logic, thresholds, or numerical outputs were intentionally changed in this revision. Existing file and directory names such as `comment5_analysis/` and `comment5_split_similarity_analysis.py` were retained to avoid breaking reproducibility paths and command examples. Their public descriptions were generalized as split/similarity audit materials.
